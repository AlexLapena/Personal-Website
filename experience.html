What did I do at Sick Kids

I started off at PhenoTips working on manual testing with test plans that I created. I was then trained in documenting bugs and issues I found in JIRA, a tool that allows software devolpers easily plan, track and release software.

Once I become more familiar with the PhenoTips software, I began writing automated tests. I found this quite a bit of fun. The tests I created used a framework called Mockito, which allowed me to mock objects in Java to create clean, readible tests. When I would finish the tests, I would submit a pull request through GitHub by the command line.

In addition to testing, I was exposed to many different things. We did several hackathons to determine if a framework like ReactJS was better than AngularJS, or we would compare different tables.
